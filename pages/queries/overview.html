Seeking after

Here you'll find all tools necessary to lazzily select particular properties in complex objects.

Think about css selectors, or jquery queries : you're asking particular DOM elements without nowing their exact place.

deepjs tries to offer same philosophy when you're working with your deeply structured objects. Simply ask, deepjs does the rest...

You could then design, manipulate or fire any parts of those complex structures without nowing exactly how they are shaped.

You'll find tools as :

  • deep query : another proposal for json/objects querying.
  • deep rql : deepjs oriented RQL js-array implementation.
  • deep traversal : traversal functions (depth or breadth first) with test callback.
  • deep selectors : small DSL that give test functions for traversal.

All those tools work with particular properties descriptors, that handle all informations on matched nodes.
