Intro

Selectors are quite experimental for the moment. This tools tries to add same kind of selection capabilities than css classes does.

Info To use it, you need to load it. 
require("deepjs/lib/selector");
Warning For the moment it use dynamic function (aka new Function(...)) and so could not be runned under paranoiac environnements (as Adobe Air). More work will be done on it in a close future.

deep.firstWith(root, selector)

Return the first property (from root) that match selector. Does not modify initial object.

var r = deep.firstWith({ e:{ b:{ sel:["hello"], a:3 } } }, "a=3");
deep.log(r); // -> { sel:["hello"], a:3 }

deep.allWith(root, selector)

Return all properties (from root) that match selector. Does not modify initial object.

var r = deep.allWith({ e:{ b:{ sel:["hello"], a:3 } }, f:{ a:3 } }, "a=3");
deep.log(r);

deep.arrayWith(root, selector, arrayName, first)

var r = deep.arrayWith({ e:{ b:{ sel:["hello"], a:3 } }, f:{ a:3 } }, "hello", "sel");
deep.log(r);
