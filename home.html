Introduction

deepjs is set of open-source isomorphic tools to manipulate javascript objects and functions. It tries first to answer modern architectural patterns problems and provides for this nothing less than a whole new way of coding.

It helps you in object, classes, views, services, routes (...) modelisation and/or to manage async runtime like a breeze.

If you searching a HTTP/Restful/Single Page/ThinServer/NoSQL framework, just part of it, or simple but so powerful tools for better architecture and modelisation, you should definitly take a look at this.

As a bit of code is better than thousand words, and even if deepjs is to large to be contained in one example...

Getting started

• Prerequired

Install nodejs then yo (it will also install grunt and bower) : 

> npm install -g yo

• Start

deepjs comes with 5 workable environements :

  • browser generator : a browser oriented minimal single page app.
  • server generator : a minimal HTTP/Restful (expressjs/deepjs) server.
  • air generator : an adobe air minimal app.
  • node-webkit generator : an desktop/nodejs oriented minimal app.
  • deep-shell : a deepjs/nodejs/shell cli and libraries (sh and ssh).

Please choose one below, install associated yo-generator, use it, and follow instructions. Each environement have it's own docs/tutorials namespace.

-- Browser --

A fully functionnal requirejs/jquery/deepjs/bootstrap environnement for browser. (full documentation.)

Install the generator globally before : 
> npm install -g generator-deepjs-browser
Then anywhere in your terminal : 
> mkdir YOUR_PROJECT_FOLDER
> cd YOUR_PROJECT_FOLDER
> yo deepjs-browser
> grunt http-server
Then go to http://localhost:8282. And follow associated tutorials.
