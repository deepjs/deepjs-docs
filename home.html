Introduction

deepjs is set of open-source isomorphic tools to manipulate javascript objects and functions. It tries first to answer modern architectural patterns problems and provides for this nothing less than a whole new way of coding.

It helps you in object, classes, views, services, routes (...) modelisation and/or to manage async runtime like a breeze.

If you searching a complete HTTP/Restful/Single Page/ThinServer/NoSQL framework, or simple tools for better architecture and modelisation, deepjs is what you need...

As a bit of code is better than thousand words...

Getting started

Prerequired

Install nodejs and yo : 

> npm install -g yo-generator-deepjs-browser
> mkdir YOUR_PROJECT_FOLDER
> cd YOUR_PROJECT_FOLDER
> yo deepjs-browser
> grunt http-server

Start

deepjs come with 4 workable environements. Please choose one below, install associated yo-generator, use it, and follow instructions. Each environement have it's own docs/tutorials namespace.

Browser

This generator provides you a full functionnal requirejs/jquery/deepjs/bootstrap environnement for browser. (full documentation.)

> npm install -g yo-generator-deepjs-browser
> mkdir YOUR_PROJECT_FOLDER
> cd YOUR_PROJECT_FOLDER
> yo deepjs-browser
> grunt http-server

Then go to http://localhost:8282. And follow associated tutorials.
